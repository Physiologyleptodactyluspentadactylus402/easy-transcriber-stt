## 🚀 What this app does

easy-transcriber-stt is a local audio transcription app for Windows. It turns speech into text on your own computer. You do not need cloud services for the main workflow. You can use OpenAI Whisper, ElevenLabs, or offline providers, based on what you want to run.

This project gives you a simple GUI and a web-style interface. It is made for people who want a direct way to transcribe lectures, meetings, voice notes, and study material.

## 💻 What you need

- Windows 10 or Windows 11
- A working microphone if you want live recording
- Audio files in common formats such as MP3, WAV, M4A, or FLAC
- Enough free disk space for the app and model files
- A modern CPU for offline transcription
- A GPU if you want faster local processing

If your computer can run a recent browser and handle media files, it should work well for basic use.

## 🛠️ Basic setup tips

To get better results:

- Use a quiet room for live recording
- Use a clear microphone

## 🧩 Troubleshooting

### The app does not open
- Check that the download finished
- Try extracting ZIP files again
- Run the EXE from a normal folder
- Check if Windows blocked the file

### The audio will not load
- Try a different file format
- Rename the file to remove odd characters
- Move the file to a simple folder path
- Check that the file is not damaged

### The transcription looks wrong
- Improve audio quality
- Pick the correct language
- Use a better microphone
- Try another provider
- Use a cleaner recording if you have one

### The app is slow
- Close other apps
- Use a smaller model if the app offers one
- Try shorter recordings first
- Run on a machine with more free memory
